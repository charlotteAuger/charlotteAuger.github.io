With Pajama Llama's Team

I worked on Flotsam during my 4 months long internship at Pajama Llama and I had the chance to be there for the game's Early Access release which took place during the last day of my internship.

During my internship I worked on new as well as already existing gameplay features, UI programming and bug-fixing during the period preceding the Early Access release.

Steam page

What I learned

First of all this internship was a very valuable experience, I feel lucky I could participate to an Early Access release and discover the linked problematics. I also learned how to adapt to a new code base and how to communicate with a team in a studio's environment.


Alternative navigation mode for agents : Flotsam agents' navigation was handled thanks to an A* pathfinding algorithm but we needed them to alternatively be able to use Unity's NavMesh navigation for specific cases. Refactoring the navigation code and getting inspiration from the Strategy design pattern to suit our needs was an interesting exercise. As my first task it also taught me how to get more familiar with the code base and find my marks.


Survival Guide : Creating an in-game guide for the player was an interesting task because it had to allow the rest of the team to easily add pages and categories without having to code. Furthermore it was my first time working on UI heavy elements and setting up a decoupled notification system for the guide was a good exercise.
