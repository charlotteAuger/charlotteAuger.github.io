Icy Skate

ID

May 2019
Game Jam

Duration : 1 week
Tools : Unity

Roles

• Programming
• Game Design

Team: Game Design : Matthieu Alves & Antoine Deprés
Game Art : Balaji Ganesh Kasibhatla & Geoffrey Boulier

Icy skate is an ultra casual mobile game prototype we developped during an intensive week in collaboration with Voodoo.io.
The player controls the ice skater with one finger the point of the game is to circle around flags and penguins to cut the ice under their feet to gain points.

Download the apk

What I learned

For this project I learned a bit about mobile optimization and practiced object pooling. Simulating the ice cutting for a cheap cost with a Depth Mask, Render order and Layer masks for the physics was also interesting to discover.
