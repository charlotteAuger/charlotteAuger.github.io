Poseidance

Game Jam Project

Duration : 48h
Tools : Game Maker

Roles

Project Leading
Programming
Game Design

Poseidance is a two player rythm game. Because of a quarrel, Poseidon challenges Mereau MacWiseau the Pirate in a dance-off.

The first player (Poseidon) chooses a set of moves, the second player must reproduce them. Of course all those moves must be done in rythm.

Playable on keyboard but gamepads are strongly advised.
