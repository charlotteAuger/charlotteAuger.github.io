Poseidance

ID

January 2017
Game Jam

Duration : 48h
Tools : Game Maker

Roles

Programming
Game Design

Poseidance is a two player rythm game. Because of a quarrel, Poseidon challenges Mereau MacWiseau the Pirate in a dance-off.

The first player (Poseidon) chooses a set of moves, the second player must reproduce them. Of course all those moves must be done in rythm.

What I learned

During this jam I had a first contact with programming a rythm game, I could also improve my knowledge of Game Maker and learned to work quickly with an adapted scope.
