snow dodge

Info

Dec. 2018
Jam Prototype

Duration : 48h
Tools : Löve 2D (Lua)






The player controls each part of the snowman with a separate key allowing it to jump and dodge projectiles thrown at the snowman.

My work

This prototype being my first experience of coding in Lua, I learned the language's basics and how to work with the Löve 2D engine during the 48h hours of the game jam.

Resources : rxi's class module.
